# File-Manager-2025

A command-line file manager built with Node.js.

## Requirements

- [Node.js](https://nodejs.org/) v18 or higher

## Installation

```bash
git clone https://github.com/amgSTRIDeR/File-Manager-2025.git
cd File-Manager-2025
npm install
```

## Usage

```bash
node src/index.js
```
